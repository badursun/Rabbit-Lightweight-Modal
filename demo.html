A lightweight, customizable modal library for modern web applications

Size Options

Choose from different modal sizes to best fit your content needs.

Example Code 
// Create a new modal with custom size
const modal = new RabbitModal({
  title: "Custom Size Modal",
  content: "Your content here",
  size: "small", // small, default, large, xlarge, cover-page
  buttons: {
    close: { text: "Close", type: "secondary" }
  }
});
modal.show();

Position Options

Position your modal anywhere on the screen with these pre-defined positions.

Example Code 
// Create a modal with custom position
const modal = new RabbitModal({
  title: "Positioned Modal",
  content: "This modal can be positioned anywhere",
  position: "top-right", // center, top, right, bottom, left, top-left, top-right, bottom-left, bottom-right
  buttons: {
    close: { text: "Close", type: "secondary" }
  }
});
modal.show();

Sidebar Options

Create sliding sidebar menus that are perfect for navigation and settings panels.

Example Code 
// Create a sidebar modal
const modal = new RabbitModal({
  title: "Sidebar Navigation",
  content: "<div class='sidebar-menu'>Your navigation content here</div>",
  position: "left-sidebar", // left-sidebar, right-sidebar
  buttons: {
    close: { text: "Close", type: "secondary" }
  }
});
modal.show();

Overlay Options

Choose from different overlay options to best fit your content needs.

Example Code 
// Create a modal with custom overlay
const modal = new RabbitModal({
  title: "Custom Overlay",
  content: "Modal with custom overlay color",
  overlay: true,
  overlayColor: "rgba(0, 0, 0, 0.8)",
  buttons: {
    close: { text: "Close", type: "secondary" }
  }
});
modal.show();

Stacked Modals

Open multiple modals on top of each other.

Example Code 
// Create multiple stacked modals
const modal1 = new RabbitModal({
  title: "First Modal",
  content: "Click the button to open another modal",
  stackable: true,
  buttons: {
    next: { 
      text: "Open Next",
      onClick: () => {
        const modal2 = new RabbitModal({
          title: "Second Modal",
          content: "This modal is stacked on top"
        });
        modal2.show();
      }
    }
  }
});
modal1.show();
